## Build & Run (Locally)
### Prerequisites
- go 1.19
- docker & docker-compose
- [swag](https://github.com/swaggo/swag) (<i>optional</i>, used to re-generate swagger documentation)


Use `make run` to run project, `make build` to run project with docker.
